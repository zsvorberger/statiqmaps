Support

This page explains how the app uses Strava data, how to connect or disconnect your account, and how to get help. It is written for Strava API review and user support.

What This App Does

This app provides personal activity summaries, historical trends, and map-based visualizations for athletes who choose to connect their Strava account. It complements Strava and does not replace or compete with it.

How Strava Data Is Used

Strava data is accessed only after user authorization and is read-only. No activities are modified or posted back to Strava. Data is shown only to the authenticated athlete.

Strava data is used solely to generate personal analytics and visualizations for the authenticated athlete.

Examples of data used include:

  • Distance, time, and elevation totals
  • Routes and activity maps
  • Activity metadata (type, date, location, gear)

What We Do NOT Do With Strava Data

  • No selling or sharing of Strava data
  • No advertising or profiling based on Strava data
  • No public display of private athlete data
  • No use of Strava data to train external machine-learning models

Connecting Your Strava Account

  • Click “Connect with Strava.”
  • Authorize access on Strava’s OAuth screen.
  • The app receives permission to read your activity data.

You can choose not to connect Strava. Some features will be limited without Strava data.

Disconnecting Strava & Data Removal

You can disconnect at any time:

  • Revoke access in your Strava account settings (My Apps).
  • Use the in-app disconnect option, if available.

After disconnecting, no further Strava data is accessed. Stored data can be deleted upon request.

Data Storage & API Usage

Data is cached responsibly to minimize API calls. Stored data is limited to the authenticated athlete. The app does not bulk scrape or make excessive requests. This app is built to respect the Strava API Agreement and Brand Guidelines.
