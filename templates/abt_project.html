About This Project

This started as a side project I built for fun when I wasn’t doing schoolwork. I like cycling, I like looking at my stats, and coding this was genuinely fun. It grew into something I actually use.

What it does

  • Dashboards that pull out trends I care about (season comparisons, big effort days, and pace changes).
  • Tools like coverage maps, route building, and basic gear tracking.
  • A simple place to check stats without digging through Strava menus.

Why I built it

Strava is great, but I wanted more control over how I view my data. This is my way to learn, tinker, and keep improving a tool I actually use.
